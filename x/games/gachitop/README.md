# Gachitop (Web)

This folder contains the web build of **Gachitop** used by `x/`.

- Runtime entry: `x/src/gachitop/launcher.js`
- WebAssembly bundle: `x/games/gachitop/gachitop.js`, `x/games/gachitop/gachitop.wasm`
- Config: `x/games/gachitop/config/`

## Build

From repo root:

```bash
bash x/games/gachitop/native/build_web.sh
```
